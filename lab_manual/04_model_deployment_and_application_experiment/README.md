# 04 Model Deployment and Application Experiment

This directory corresponds to the `Model Deployment and Application Experiment` section of the HCIA-AI V4.0 Lab Framework.

## Structure

```text
04_model_deployment_and_application_experiment/
├── 01_cli_based_llm_deployment/
└── 02_web_based_llm_deployment/
```

## Mapping

| No. | Experiment | Description |
|---|---|---|

| 02 | Web-based LLM Deployment | Start `llama-server` and call health, completion, and OpenAI-compatible APIs from Python |
